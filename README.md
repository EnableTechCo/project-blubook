# BluBook Frontend Scaffold

Phased Next.js + Supabase frontend scaffold for the BluBook multi-role operations platform.

## Stack

- Next.js (App Router)
- TypeScript (strict)
- Tailwind CSS
- Supabase (Auth, Postgres, Realtime, Storage)
- Zustand + React Query

## Quick Start

1. Install dependencies:
   npm install
2. Copy environment template:
   cp .env.example .env.local
3. Fill Supabase credentials in `.env.local`.
4. Start dev server:
   npm run dev

## Environment

Required variables are listed in `.env.example`.
